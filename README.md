# Omok AI Game

🎯 HTML + JavaScript로 만든 완전한 오목 게임  
🤖 4단계 난이도 AI (초급 ~ 전문가)  
💡 힌트 시스템: 최선의 수 표시  
✨ 모던 그라디언트 UI 디자인  
🎮 19x19 바둑판 + 화점 표시  

## 실행 방법
1. `index.html` 파일을 브라우저에서 열기
2. [혹은 여기를 클릭해서 실행하기](https://mahyun-dev.github.io/omok-game/index.html)

## 조작법
- **마우스 클릭**: 돌 놓기
- **힌트 버튼**: 최선의 수를 초록색으로 표시 (3초간)
- **무르기 버튼**: 2수 되돌리기 (플레이어 + AI)
- **새 게임 버튼**: 게임 초기화

## 주요 기능
### 🎯 완전한 오목 규칙
- 5개 연속으로 승리
- 6목 규칙 적용 (6개 이상 연결 시 무효)
- 19x19 표준 바둑판
- 화점 표시 (9개 지점)
- 마지막 수 빨간 테두리 표시

### 🤖 AI 난이도
1. **초급**: 랜덤 배치
2. **중급**: 2단계 깊이 탐색
3. **고급**: 3단계 깊이 탐색
4. **전문가**: 4단계 깊이 탐색

### 💡 AI 전략
- **즉시 승리 체크**: 5목 만들 수 있으면 즉시 실행
- **방어 체크**: 상대방 5목 막기
- **패턴 분석**: 열린 4, 막힌 4, 열린 3 등 평가
- **중앙 가중치**: 초반에 중앙 선호
- **알파-베타 가지치기**: 탐색 효율성 향상

### 🎨 UI 특징
- 그라디언트 배경 (보라 → 핑크)
- 부드러운 애니메이션 효과
- 반응형 디자인 (모바일 지원)
- 글래스모피즘 효과

## 파일 구조
```
├── index.html          # 메인 HTML 구조
├── styles.css          # 스타일시트
├── game.js             # 게임 초기화 및 렌더링
├── validation.js       # 오목 규칙 검증 (5목, 패턴 분석)
├── actions.js          # 돌 놓기, 무르기
└── ai.js               # AI 로직 (Minimax 알고리즘)
```

## 기술 스택
- **HTML5**: 시맨틱 마크업
- **CSS3**: 그라디언트, 애니메이션, Flexbox/Grid
- **Vanilla JavaScript**: 순수 자바스크립트로 구현
- **알고리즘**: Minimax with Alpha-Beta Pruning

## AI 알고리즘
### Minimax 알고리즘
- 상대방의 최선의 수를 고려한 의사결정
- Alpha-Beta Pruning으로 불필요한 탐색 제거
- 깊이 제한으로 성능 최적화

### 평가 함수
패턴별 점수:
- **5목**: 100,000점 (승리)
- **열린 4**: 10,000점 (다음 턴 승리 확정)
- **막힌 4**: 1,000점
- **열린 3**: 500점
- **막힌 3**: 100점
- **열린 2**: 50점
- **막힌 2**: 10점

추가 평가:
- 중앙 위치 보너스 (초반 전략)
- 연속된 돌의 방향성 분석 (가로, 세로, 대각선 2개)

## 최적화 기법
- **가지치기**: 평가 점수 상위 10~20개만 탐색
- **근접 탐색**: 이미 놓인 돌 근처만 고려
- **즉시 승리/방어**: 깊이 탐색 전 우선 체크
- **첫 수 최적화**: 중앙에 배치

## 게임 규칙
1. 흑돌(플레이어)이 먼저 시작
2. 교대로 돌을 놓음
3. 5개를 먼저 연결하면 승리
4. 6개 이상 연결 시 무효 (렌주룰 일부 적용)
